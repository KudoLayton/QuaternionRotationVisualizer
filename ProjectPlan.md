# 프로젝트 개요
- 이 프로젝트는 3차원 회전을 Quaternion으로 서술하는 이유를 독창적인 시각화를 실험하는 것입니다.
- 다음 사항을 증명할 수 있을지 실험해보고 싶습니다.
    - (x, y, z) 축을 기준으로 할 때 왜 회전이 cos(\theta) + u * sin(\theta) 으로 기술되는지
        - u = x * i + y * j + z * k
    - 왜 QvQ^(-1) 로 서술해야하는 지

# 접근 방법
## multiplication의 기하학적 의미
a * b 라는 곱셈이 있을 때 이를 다음과 같이 해석 합니다.
1. 곱셈의 항등원인 1 과 a가 있는 좌표계가 있음
2. 1 을 b가 되도록 좌표계를 변환함
3. 변환된 좌표계 내에서 기존 a가 어느 값으로 변경되었는지 확인

## 2차원에서는 안그랬는데 3차원 좌표계 회전을 다룰 때 왜 새로운 차원이 필요한가?
- 일반적인 2 차원 평면 좌표계에서 회전은 2 차원을 구성하는 축이 아닌 평면 좌표계와 완전히 수직한 축이 회전 축임
- 2차원 평면 위의 직선을 회전축으로 잡으면 이 역시 "법선" 방향의 정보가 필요함
- 3차원 좌표계 역시 3차원 좌표계로 서술된 회전축을 사용하여 추가 차원을 요구한 것임

# 전략
- 3차원 회전이 아닌 2차원의 회전을 다뤄보는 것
    - 여기서 2차원은 x, y축 평면을 이야기 함
- 3차원 입장에서의 추가 차원 (quaternion의 실수부) 을 2차원에서는 z 축으로 생각할 것
- 원점을 지나는 평면위의 회전축을 정의 (방향 벡터 - D)
- 회전 대상이 될 평면위의 임의의 점 V
- \theta 만큼 회전해보는 것

## 유사 quaternion 회전 적용
- 유사 quaternion Q를 정의 
    - Q = cos(\theta) * z^(\hat) + sin(\theta) * D
- quaternion에서는 (1, 0, 0, 0)이 항등원이기 때문에 유사하게 이번에는 항등원을 z^(\hat) 으로 생각함 (단위 z축 벡터)
    - 당연히 대수적으로 항등원이 절대 아니나 기하학적인 의미를 실험해보고 싶어서임

## 연산
1. QV 연산 - multiplication의 기하학적 의미 대로 진행
    1. 단위 z축 벡터와 Q가 있는 좌표계가 있음
    2. 단위 z축 벡터를 v 위치로 좌표계 전체 선형 변환
    3. 변환된 Q 위치인 _Q 확인
2. QVQ^(-1) 연산
    1. 단위 z축 벡터와 _Q가 있는 좌표계가 있음
    2. 단위 z축 벡터가 Q^(-1) 위치로 이동하도록 좌표계 전체 선형 변환
    3. 변환된 _Q 위치 확인

# 주요 요구 사항
- 각 연산 과정을 부드러운 애니메이션 형태로 보여줬으면 좋겠음
- 3D viewport를 일반 3D 그래픽 도구처럼 마우스로 회전, 확대, 이동 등이 되었으면 좋겠음
